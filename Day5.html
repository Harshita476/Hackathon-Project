HTML Media

HTML provides several tags for embedding and controlling media on a webpage. These tags allow for the inclusion of images, audio, and video, making web content more interactive and engaging.

Media Tags :D

  • <audio>

    This tag embeds audio content, such as music or sound effects.

    • source: allows you to specify multiple media files.
    • autoplay: instructs the browser to automatically begin media playback as soon as the page is loaded.

  • <video>

    The video HTML element embeds a media player which supports video playback into the document.

  • <source>

    The <source> HTML element specifies one or more media resources for the <picture>, <audio>, and <video> elements.

  • <iframe>

    This tag specifies an inline frame. An inline frame is used to embed another document within the current HTML document.
