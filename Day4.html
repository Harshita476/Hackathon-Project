What is a Form ??

The <form> HTML element represents a document section containing interactive controls for submitting information.They incorporate a variety of interactive controls such as text fields, numeric inputs, email fields, password fields, checkboxes, radio buttons, and submit buttons.

An <input> element can be displayed in many ways, depending on the type attribute.

Type Description
<input type="text"> Displays a single-line text input field
<input type="radio"> Displays a radio button (for selecting one of many choices)
<input type="checkbox"> Displays a checkbox (for selecting zero or more of many choices)
<input type="submit"> Displays a submit button (for submitting the form)
<input type="button"> Displays a clickable button

Form Tags :D

  • <form> : This is the container tag for all form elements. It defines the start and end of the form.

  • <input> : This is a versatile self-closing tag used for various types of user input.

  • <label> : Provides a caption for a form element, improving accessibility and user experience.

  • <button> : Creates a clickable button. It can be used for submitting the form

  • <textarea> : Defines a multi-line text input control, suitable for comments or longer text entries. It uses rows and cols attributes to define its size.

  • <select> : Creates a drop-down list (dropdown menu). It contains one or more <option> tags.

  • <option> : Groups related options within a <select> drop-down list, providing a hierarchical structure.
